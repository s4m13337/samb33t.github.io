X Sin(x) = X-Mas Tree

A christmas tree animation with Mathematica

We have a line ...

Plot[x, {x, 0, 30}]

And we have a sine wave ...

Plot[x Sin[x], {x, 0, 30}]

Then we have the product of the line and the sine wave ...

Plot[x Sin[x], {x, 0, 30}]

And then we have its negative ...

Plot[-x Sin[x], {x, 0, 30}]

Now lets see them together ...

Show[
Plot[x Sin[x], {x, 0, 30}],
Plot[-x Sin[x], {x, 0, 30}],
PlotRange -> Full
]

Its time to get rid of the axes ...

Show[
Plot[x Sin[x], {x, 0, 30}],
Plot[-x Sin[x], {x, 0, 30}],
PlotRange -> Full,
Axes -> False
]

And rotate every thing by 90 degrees ...

Rotate[
Show[
Plot[x Sin[x], {x, 0, 30}],
Plot[-x Sin[x], {x, 0, 30}],
PlotRange -> Full,
Axes -> False
], -90 \[Degree]
]

Adding a few traditional Christmas colors ...

Rotate[
Show[
Plot[x Sin[x], {x, 0, 30}, PlotStyle -> {Red, Thick}],
Plot[-x Sin[x], {x, 0, 30}, PlotStyle -> {Darker@Green, Thick}],
Axes -> False
], -90 \[Degree]
]

And a background ...

Rotate[
Show[
Plot[x Sin[x], {x, 0, 30}, PlotStyle -> {Red, Thick}],
Plot[-x Sin[x], {x, 0, 30}, PlotStyle -> {Darker@Green, Thick}],
Axes -> False,
Background -> Black
], -90 \[Degree]
]

A bit of styling ...

Rotate[
Show[
Plot[x Sin[x], {x, 0, 30}, PlotStyle -> {Red, Thick, Dashed}],
Plot[-x Sin[x], {x, 0, 30}, PlotStyle -> {Darker@Green, Thick, Dashed}],
Axes -> False,
Background -> Black
], -90 \[Degree]
]

Some snowflakes. We are almost there ...

Overlay[{
Rotate[
Dynamic[ListPlot[
Table[{RandomReal[]*10, RandomReal[]*10}, {i, 1, 20}],
PlotMarkers -> Style["*", 18],
Background -> Black,
Axes -> False,
PlotStyle -> White]
], -90 \[Degree]],
Rotate[
Show[
Plot[x Sin[x], {x, 0, 30}, PlotStyle -> {Red, Thick, Dashed}],
Plot[-x Sin[x], {x, 0, 30}, PlotStyle -> {Darker@Green, Thick, Dashed}],
Axes -> False,
Background -> Transparent
], -90 \[Degree]
]
}]
